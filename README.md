# 🚗 Car Price Prediction using Machine Learning

## 📌 Overview

This project predicts the price of used cars using Machine Learning techniques. The model is trained on a dataset containing car information such as company, year, fuel type, and kilometers driven.

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Linear Regression

---

## 📊 Dataset

The dataset contains the following features:

* Car Name
* Company
* Year
* Price
* Kilometers Driven
* Fuel Type

---

## ⚙️ Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Training
6. Price Prediction

---

## 🤖 Machine Learning Model

* Linear Regression
